# emg112-apis

> Express(Node.js) + MongoDB Backend for REST APIs of Emergency 112

## Prerequisites

```yml
# Digital Ocean
"node.js": "14.21.3"
"npm": "6.14.18"

# process management
"pm2": "5.3.0"
```

## How To Run

```bash
$ npm install

$ npm start
```

Visit http://localhost:8080/

## Admin Panel

`adminpanel` folder is for admin page using React.

Build and use it for Nginx public html.
